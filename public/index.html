Create a React Vite App
Create a React app using the create vite command.
Choose the default option for every prompt (hit Enter).
npm create vite @ latest my - app
Install the Lezzserver Library
To get started, install the lezzserver package which provides a convenient interface for working with Lezzserver from a React app

Navigate to your app directory and install lezzserver
cd my-app && npm install lezzserver
Setup a Lezzserver Dev Deployment
Next, run npx lezzserver dev. This will prompt you to log in with GitHub, create a project, and save your production and deployment URLs.

It will also create a lezzserver/ folder for you to write your backend API functions in. The dev command will then continue running to sync your functions with your dev deployment in the cloud.
npx lezzserver dev
